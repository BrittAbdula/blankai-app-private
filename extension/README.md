# BlankAI Extension

Standalone browser extension build for Chrome and Edge.

## Commands

```bash
cd extension
pnpm run dev
pnpm run build
pnpm run check
```

After `pnpm run build`, you can load either of these as an unpacked extension:

- `/Users/tm/Desktop/blankAI/extension`

## Store Upload

Build first, then upload a zip of `/Users/tm/Desktop/blankAI/extension`.
Do not upload `/Users/tm/Desktop/blankAI/extension/dist` directly.
